SAFEBITES FULLSTACK

Cara menjalankan backend:

1. cd backend
2. npm install
3. npm start

Frontend:
Buka frontend/index.html

Admin:
Buka admin/dashboard.html

Database:
Import database/safebites.sql ke phpMyAdmin

Midtrans:
Isi SERVER KEY dan CLIENT KEY di:
backend/config/midtrans.js

npm install midtrans-client

npm install dotenv

npm install multer
